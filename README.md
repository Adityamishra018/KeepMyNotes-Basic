# KeepMyNotes

## About the Project

Note keeping app build using Express/Mongo and EJS (Server side rendered).
The application has session based authentication and has been implemented in VanillaJS.

## Concepts Covered

- Session Based Auth
- MongoDB
- Express
- EJS
